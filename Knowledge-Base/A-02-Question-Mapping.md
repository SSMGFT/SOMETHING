
# Appendix A-02 - Question to Concept Mapping

> Version: 1.5

# Purpose

Do **not** memorize question numbers.

Use each question to identify the underlying concept(s). If you miss a question,
review the listed concepts rather than rereading the whole domain.

---

## Mapping Strategy

```
Question
    │
    ▼
Concept(s)
    │
    ▼
Module
    │
    ▼
Review
```
